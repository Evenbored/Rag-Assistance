# Rag-Assistance

Пет-проект RAG-ассистента. Работает как тг-бот, документы может заливать только администратор. 

## Как это устроено внутри

Проект полностью написан на Python. Из библиотек я взял FastAPI для веб-сервера, aiogram для Telegram-бота, а связку SQLAlchemy, Alembic и asyncpg для обычной базы данных и миграций. Нейросетевая часть и векторный поиск работают на PyTorch, Transformers и расширении pgvector.
